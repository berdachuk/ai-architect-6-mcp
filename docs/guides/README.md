2. **Tuning prompts?** Read [prompt-lab-user-guide.md](prompt-lab-user-guide.md) after the MCP guide.
3. **Local smoke test?** Use [claude-desktop-mcp.md](claude-desktop-mcp.md) or [lm-studio-mcp-manual-test.md](lm-studio-mcp-manual-test.md) with Docker Compose.
