# DEC-009 — WSL for Docker on Windows

| Field | Value |
|---|---|
| Status | Accepted |
| Date | 2026-06-16 |
| Source | Dev environment note |

On Windows dev machines, use **WSL** to access Docker (Docker Desktop WSL2 backend). Run `docker` CLI, `docker compose`, and `mvn verify -Pintegration` (Testcontainers) from a **WSL shell** — not native PowerShell/CMD. Clone or work under the WSL filesystem (`~/projects/...`) when possible for better I/O.
