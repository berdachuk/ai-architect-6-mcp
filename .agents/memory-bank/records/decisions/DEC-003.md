# DEC-003 — Mandatory EmbeddingEndpointPool

| Field | Value |
|---|---|
| Status | Accepted |
| Date | 2026-06-16 |
| Modules | `embedding`, `dataset` |
| Source | [docs/01-requirements.md §4](../../../docs/01-requirements.md#4-embedding-model) |

All embedding traffic routes through `EmbeddingEndpointPool` — always on, never optional. At least one endpoint with a non-blank `url` is required; startup fails fast if the pool would be empty. No Spring AI embedding auto-configuration (`OpenAiEmbeddingAutoConfiguration` excluded). `OpenAiEmbeddingModel` instances built manually per endpoint.
