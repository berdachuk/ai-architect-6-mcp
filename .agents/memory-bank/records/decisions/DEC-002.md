# DEC-002 — JDBC only, no JPA

| Field | Value |

Persistence uses `NamedParameterJdbcTemplate` throughout. No JPA/Hibernate, no `@Entity`, no `ddl-auto`. Schema is Flyway-managed. Repository impls live in `*/repository/impl/*` with `@Repository`.
